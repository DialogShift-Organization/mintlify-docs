---
title: "Bookolo Integration"
description: "How to use Bookolo with DialogShift for availability and rates in chat and on the phone."
---

The Bookolo integration connects DialogShift to Bookolo for availability and rate lookup. Journey Messaging and checkout through DialogShift are not supported for Bookolo.

## Prerequisites

- Your hotel uses Bookolo.
- You have the Bookolo credentials.

## Activate the integration

Because there is a Bookolo integration page in the DialogShift app, the hotel can activate the integration itself.

1. Enter the Bookolo credentials in the DialogShift app.
2. Activate Bookolo as the booking engine.
3. Check availability and rates in the DialogShift Availability Analyzer.

## Booking agents

| Topic | Behavior |
|:------|:---------|
| Company code / coupons / discount codes | Not supported. |
| Minimum stay in the dialog | Not explicitly communicated by DialogShift. |
| Checkout workflow | No DialogShift checkout. |
| Payment link in chat | Not supported. |
| Payment link by phone | Not supported. |
| Cancellation of unpaid bookings | Not applicable because DialogShift does not create a Bookolo booking. |
| Payment variants | Not bookable through DialogShift. |
| Offer/quote function | No. |
| Required guest data | None in DialogShift checkout because there is no DialogShift checkout. |

## Booking agent configuration

**API credentials.** Bookolo credentials are configurable so DialogShift queries availability and rates for the correct hotel. Enter the credentials in the DialogShift app and then verify a representative request in the Availability Analyzer.

**Room and rate enablement.** Rooms and rates are configurable so the agent only shows public offers. Do not enable internal, group or special rates that are not meant for guests in the dialog.

## Journey Messaging

Bookolo is not currently supported as a PMS integration for Journey Messaging. To run automated journey flows, you need a supported PMS integration.

## Configuration

The technical Bookolo connection is maintained in the DialogShift app.
