---
title: "Booking Engine Integrations"
description: "Which integrations DialogShift uses for availability, rates, bookings and checkout links in Chat AI and Phone AI."
---

Booking engine integrations connect DialogShift to your hotel's booking flow or PMS. Chat AI and Phone AI use them for availability, rates and, depending on the provider, bookings or checkout links.

| System | Booking agent status | Checkout | Activation | Details |
|:-------|:---------------------|:---------|:-----------|:--------|
| apaleo | Booking supported | Direct booking + apaleo payment link when card details are required | Hotel activates it in the DialogShift app | [apaleo Integration](/en/integrations/apaleo-integration) |
| Bookolo | Availability/rates only | No DialogShift checkout | Hotel activates it in the DialogShift app | [Bookolo Integration](/en/integrations/bookolo-integration) |
| Casablanca | Availability/rates only | No DialogShift checkout | Customer Success activates the engine | [Casablanca Integration](/en/integrations/casablanca-integration) |
| DIRS21 | Availability/rates only | No DialogShift checkout | Hotel activates it in the DialogShift app | [DIRS21 Integration](/en/integrations/dirs21-integration) |
| Ennismore / 25hours | Availability/rates only | No DialogShift checkout | Customer Success activates the engine | [Ennismore / 25hours Integration](/en/integrations/ennismore-integration) |
| HNS | Booking via link | HNS OnePageBooking link | Hotel activates it in the DialogShift app | [HNS Integration](/en/integrations/hns-integration) |
| HotelPartner | Booking supported | Direct booking or offer link | Hotel activates it in the DialogShift app | [HotelPartner Integration](/en/integrations/hotelpartner-integration) |
| MEWS | Booking supported | Direct booking + MEWS payment request | Hotel maintains the configuration in the DialogShift app | [MEWS Integration](/en/integrations/mews-booking-agent-go-live) |
| Oracle OHIP | Availability/rates only | No DialogShift checkout | Customer Success activates the engine | [Oracle OHIP Integration](/en/integrations/oracle-ohip-integration) |
| Protel / Planet | Booking supported | Direct pay-at-hotel booking | Hotel activates it in the DialogShift app | [Protel / Planet Integration](/en/integrations/protel-planet-integration) |
| Samera | Booking via link | Samera checkout link | Customer Success activates the engine | [Samera Integration](/en/integrations/samera-integration) |
| Workmatrix | Availability/rates only | No DialogShift checkout | Customer Success activates the engine | [Workmatrix Integration](/en/integrations/workmatrix-integration) |

Journey Messaging is a separate capability. Supported PMS integrations are listed in the [Journey Messaging integrations overview](/en/integrations/journey-messaging-integrations).
