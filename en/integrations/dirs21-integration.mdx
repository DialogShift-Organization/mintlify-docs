---
title: "DIRS21 Integration"
description: "How to use DIRS21 with DialogShift for availability and rates in chat and on the phone."
---

The DIRS21 integration connects DialogShift to your DIRS21 One booking engine. Chat AI and Phone AI can retrieve availability, prices and rates. Journey Messaging and checkout through DialogShift are not supported for DIRS21.

## Prerequisites

- Your hotel uses DIRS21 One.
- The integration is enabled with DIRS21 (paid service at DIRS21).
- Your DialogShift Client ID (in the app under **Settings → Integration**).

## Activate the integration

Because there is a DIRS21 integration page in the DialogShift app, the hotel can activate the integration itself once DIRS21 has provided the credentials.

1. Commission the DialogShift integration with DIRS21.
2. DIRS21 sends the credentials to you or directly to DialogShift.
3. Enter the values in the DialogShift app under **Settings → Integration**.

## Booking agents

| Topic | Behavior |
|:------|:---------|
| Company code / coupons / discount codes | Not supported. |
| Minimum stay in the dialog | Not explicitly communicated by DialogShift. |
| Checkout workflow | No DialogShift checkout. DIRS21 is used for availability and rates. |
| Payment link in chat | Not supported. |
| Payment link by phone | Not supported. |
| Cancellation of unpaid bookings | Not applicable because DialogShift does not create a DIRS21 booking. |
| Payment variants | DIRS21 can report whether a credit card is required; checkout does not happen through DialogShift. |
| Offer/quote function | Offers are used only for rates/prices. |
| Required guest data | None in DialogShift checkout because there is no DialogShift checkout. |

## Journey Messaging

DIRS21 is not currently supported as a PMS integration for Journey Messaging. To run automated journey flows, you need a supported PMS integration.

## Configuration

The following fields are provided by the DIRS21 support team and entered in the DialogShift app:

- **Channel ID** – the connection identifier for DIRS21.
- **DIRS21 directory** (optional) – only needed if DIRS21 runs a separately evaluated copy of the booking engine with the same prices and availability.
- **Booking Engine Version** – preset; usually leave unchanged (choose ONE if in doubt).
