</Note>

## Requirements

- A DialogShift account
- Exchange Server (Exchange Online or Exchange 2016+)

<Note>
IMAP accounts are not supported.
</Note>

### Compatible Outlook Versions

**Windows:**
- Outlook 2021 or newer
- Outlook 365 on Windows 10 20H2 or newer
- New Outlook

Not supported: Outlook 2019 or older; Outlook 365 that is not regularly updated or running on Windows 10 2004 or older.

**Mac:**
- Outlook App & Outlook 2024

Not supported: Outlook with a year number (except 2024).

**Web:**
- Outlook online (Firefox, Safari, Chrome and derivatives)

Not supported: Outlook Web Access.

## Installation

### With Exchange Online

1. Start Outlook on your computer
2. Click **All Apps** in the top right and select **Get Add-ins**
3. Enter **DialogShift** in the search field
4. Select **DialogShift Email Assistant** and click **Add**
5. The plugin is now available in the **Add-ins** tab under **My Add-ins**

### With Exchange On-Premise

1. Start the Outlook web interface in your browser and sign in with an account that has sufficient permissions
2. Navigate to the overview of all installed apps
3. Select the option to add a new plugin via URL and enter:

```
https://store.office.com/app/download?assetid=WA200006927&cmu=en-001&ClientSessionId=9e50467e-3598-4440-b754-797ccc61f5b1
```
