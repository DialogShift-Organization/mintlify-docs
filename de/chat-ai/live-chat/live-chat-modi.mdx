DialogShift bietet drei verschiedene Modi für die Live Chat Funktion, um die Kommunikation mit deinen Gästen optimal zu gestalten.

<CardGroup cols={3}>
  <Card title="Keine Live Chats" icon="robot">
    Vollautomatischer Modus - nur Chatbot AI
  </Card>
  <Card title="Bürozeiten" icon="clock">
    Live Chat nur während definierter Zeiten
  </Card>
  <Card title="Immer erlauben" icon="circle-check">
    24/7 mit automatischem Abwesenheitshinweis
  </Card>
</CardGroup>

## Einstellungen aufrufen

Navigire zu: **Einstellungen \>\> Live Chat**

## Modi im Detail

### 🤖 Keine Live Chats

<Info>
  Ideal für Hotels, die vollständig auf KI-Automatisierung setzen möchten.
</Info>

- Chatbot AI beantwortet alle Anfragen automatisch
- Keine manuellen Eingriffe durch Mitarbeiter
- Alternative Kontaktmöglichkeiten werden automatisch angeboten

**Aktivierung:** Wähle im Dropdown **"Keine Live Chats"**

### 🕐 Live Chats nur während der Bürozeiten

<Note>
  Perfekt für Hotels mit festen Rezeptionszeiten.
</Note>

- Live Chat nur innerhalb definierter Zeitfenster verfügbar
- Bis zu zwei Zeitfenster pro Tag möglich
- Außerhalb der Zeiten übernimmt der Chatbot

**Aktivierung:**

1. Wähle **"Live Chats nur während der Bürozeiten"**
2. Definiere deine Zeitfenster
3. Speichere die Einstellungen

### ✅ Live Chats immer erlauben

<Tip>
  Bietet maximale Flexibilität für Gäste bei gleichzeitiger Transparenz.
</Tip>

- Gäste können jederzeit einen Live Chat anfordern
- Automatischer Abwesenheitshinweis außerhalb der Bürozeiten
- Information über nächste verfügbare Live Chat Zeit

**Aktivierung:**

1. Wähle **"Live Chats immer erlauben (mit Abwesenheitshinweis)"**
2. Stell deine Live Chat Zeiten ein
3. Speichere die Konfiguration
