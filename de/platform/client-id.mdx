## Was ist die Client ID?

Die **Client ID** identifiziert dein Konto/Hotel in DialogShift eindeutig und wird für **Integrationen mit Partnerunternehmen** verwendet.

## Wo finde ich die Client ID?

Du findest die Client ID direkt in der **URL** der DialogShift App.

<img
  src="/images/client-id-url.png"
  alt="Client ID in der URL der DialogShift App"
/>

Sie:

- beginnt immer mit **`pro`**
- hat danach **4 Zeichen** (Buchstaben oder Zahlen)

### Beispiel

Wenn deine URL so aussieht:

`https://app.dialogshift.com/pro1c75/hotel/settings/general`

Dann ist die **Client ID**:

`pro1c75`

## Wofür wird sie genutzt?

Gib die Client ID an Partnerunternehmen (oder an den DialogShift Support) weiter, wenn eine Integration dem richtigen DialogShift-Client zugeordnet werden soll.
